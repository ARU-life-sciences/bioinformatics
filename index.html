Using the terminal

The most familiar use of a terminal is through a command line interface (or CLI). Using a CLI, we can type commands, and the computer will print results back to the screen, modify our system, or write to a file. Our introduction covers installation of terminals, file and file system manipulation, fetching data, and downloading programs.

Using R

R is a popular programming language in the biological sciences. Whilst not recommended for writing efficient programs for large genomic data sets, it is very useful for manipulating tabular data and plotting. We introduce the language here.

Statistics in R

Bioinformatics still leans heavily on statistics for analysis of data. Here we run through all major and important statistical tests, which are useful not just in bioinformatics, but across the biological sciences.
