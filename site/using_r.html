Using R

R is a very popular programming language, especially amongst data scientists and academics. It's widely used in industry. Built largely for statistical programming in the late 1990's and early 2000's, R has expanded in functionality to almost all realms of programming - from mathematical computing to building websites. Here we present a short format introduction to R. We introduce background data types but quickly get in to reading in and manipulating data. If this is your first time here, start at the top and make your way to the bottom.

Variables, data types, and classes

Variables are the building blocks of programs. Using assignment, we can assign values to arbitrary names, and reference them elsewhere in our programs. We introduce how to do this, along with understanding the basic data types in R. It is possible to define our own data types too using classes.

Cheat Sheets

Some useful cheat sheets for various topics we have covered.

  • The R reference card has lots of information on base R syntax and functions. Of special interest might be the operators (logical and otherwise), as well as all the plot parameters in base `plot`
  • The R base cheat sheet has much similar information to the above but is perhaps simpler.
  • The one page reference card is a single page of commonly used and useful R snippets.
  • If you use RStudio, which most people do now, then the RStudio cheat sheet is an excellent resource. It tells you what all the panels and buttons do.
  • Using ggplot is a main-stay for many R programmers, but the syntax is slightly unintuitive, and the grammar of graphics can be tricky to remember. Use the ggplot2 cheat sheet to refresh your memory.
